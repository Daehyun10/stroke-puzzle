# 한자 획순 자물쇠 퍼즐

조선시대 방탈출 게임을 위한 한자 획순 인식 퍼즐 시스템.

## 기능
- 마우스 드래그로 획 입력 인식
- 8방향 획 방향 분류 (→ ← ↑ ↓ ↗ ↖ ↘ ↙)
- 획순 정답 검증
- 정답 피드백 (금색 선 + 파티클 이펙트)
- 오답 피드백 (빨간 선 + 페이드아웃 + 흔들림)
- ScriptableObject 기반 한자 데이터 구조

## 사용 스택
- Unity 6 (2D)
- C#

## 구현 완료
- [x] StrokeDirection 열거형 (8방향)
- [x] HanjaData ScriptableObject (한자 데이터 구조)
- [x] 한자 에셋 3종 (門, 一, 水)
- [x] StrokeInputHandler (마우스 드래그 인식)
- [x] 벡터 → 8방향 변환 로직
- [x] StrokeValidator (획순 검증)
- [x] StrokeDrawer (LineRenderer 획 시각화)
- [x] 정답 피드백 (금색 선 애니메이션 + 파티클)
- [x] 오답 피드백 (빨간 선 페이드아웃 + 흔들림)

- [ ] 힌트 시스템 (3단계)

## 한자 데이터 예시
```
門 (문 문) : ↓↓→↓↓→↓↓
一 (한 일) : →
水 (물 수) : ↓↙↘↓
```
